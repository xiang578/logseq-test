使用场景

  + 数组会被分割成若干段，且每一段判断/处理逻辑是一样的。

好处

  + 无序判空

  + 无需在循环结束后再补上最后一段区间的逻辑

```python
i, n = 0, len(nums)
while i < n:
    start = i
    while i < n and ...:
        i += 1
    # 从 start 到 i-1 是一段
    # 下一段从 i 开始，无需 i+=1
```
