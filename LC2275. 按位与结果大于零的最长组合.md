## 题解

  + 标准贪心题目，按位与最后结果要大于 1，对应选择的数字在某一个二进制位上都是 1。所以反过来统计每一个二进制位上为 1 的数字有几个，取最大就是最后的答案。

## 参考代码

```python
class Solution:
    def largestCombination(self, candidates: List[int]) -> int:
        
        ans = 0
        for i in range(32):
            tmp = 0
            now = 1 << i
            # print(i, now)
            for c in candidates:
                if c & now > 0:
                    tmp += 1
            ans = max(tmp, ans)
        return ans
```
