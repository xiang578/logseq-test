## 题解

## 参考代码
```python
#
# @lc app=leetcode.cn id=3019 lang=python3
#
# [3019] 按键变更的次数
#

# @lc code=start
class Solution:
    def countKeyChanges(self, s: str) -> int:
        ans = 0
        s = s.lower()
        for i in range(1, len(s)):
            if s[i] != s[i-1]:
                ans += 1
        return ans
# @lc code=end

```
